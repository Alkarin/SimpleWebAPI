## Summary

A bare bones Android Application, used to display the technical feasibility and basics of accessing a web API from an Android application.

This repository is based off the code and comments mention in this article:

https://www.androidauthority.com/use-remote-web-api-within-android-app-617869/
### How to use:

* Input an email address and hit `search` to send a request to the web API to return information on the email address.

That's it!



## Installation:


Simple Android Application, build onto your device via an IDE or build the APK and install.
